## 📖 Overview

The resale value of a used vehicle depends on multiple factors such as **vehicle age, fuel type, transmission, mileage,** and **brand**. Manually estimating prices can often be inaccurate and time-consuming.

This project uses a trained Machine Learning regression model to analyze vehicle-related features and provide accurate resale price predictions through an interactive web interface built with **Streamlit**.

## ⚙️ Installation & Setup

```bash

```

### 2️⃣ Navigate to the Project Directory
```bash

```

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application
