🚗 Used Vehicle Price Predictor

An AI-powered web application that predicts the resale price of used vehicles using Machine Learning techniques and Gradient Boosting Regression.

📖 Overview

The resale value of a used vehicle depends on multiple factors such as vehicle age, fuel type, transmission, mileage, and brand. Manually estimating prices can often be inaccurate and time-consuming.

This project uses a Machine Learning regression model to analyze vehicle-related features and provide accurate resale price predictions through an interactive web interface built with Streamlit.

🎯 Project Objectives
Predict the resale price of used vehicles accurately
Build a real-world Machine Learning application
Develop an interactive and user-friendly web interface
Apply data preprocessing and feature engineering techniques
Deploy an AI-powered prediction system using Streamlit

User enters vehicle details
Input data is preprocessed
Trained Gradient Boosting model predicts resale price
Predicted price is displayed instantly on the web interface
